Fixed Design

Fixed design allows you to create a fixed page size.

It will look the same on all browsers however it will not shrink to the size of the device. Fixed designs are used for business, photography and artist sites when design concepts are important.

Pros
  • Full HTML layout options available.
  • Fast drag and drop editing.
  • Does not shrink and change layout.
  • Looks exactly like you build it.

Cons
  • Does not shrink to fit a mobile device.

I use themes that contain Side Bars.
Responsive Design
Responsive pages adjust to the browsers size and typically push the content down as the page width becomes smaller.

Plug N Edit uses scripting to adjust the size, instead of just pushing the content down. Compatible with all modern browsers except EI8, a full page version is displayed in that case .

Pros
  • One page fits all.
  • Full HTML layout options still available.
  • Does not change layout.
  • Looks exactly like you build it

Cons
  • You must use larger text sizes so they display in mobile devices well.
I use themes that contain Side Bars.
Responsive Adaptive Design

Adaptive design allows you to create a mobile version and a full sized version of your page. This is for people who want full control of the look of their website on mobile devices and desktops.

The Adaptive Responsive Design will also adjust the fit of the screen.

Pros
  • Custom design your mobile pages.
  • Full HTML layout options still available.
  • Does not change layout.
  • Looks exactly like you build it

Cons
  • You must build 2 individual layouts one mobile, one full size.
I use themes that contain Side Bars.
